[id="introduction"]
== Introduction

Fidget is a Kotlin Multiplatform library for building cross-platform mobile applications with Compose Multiplatform.  
This library targets both Android and iOS, providing shared UI components and logic via KMP.

[id="installation"]
== Installation

=== Requirements

* Java 17+
* Kotlin 1.9+
* Gradle 8.0+
* (For Android): Android SDK with compileSdk 36+
* (For iOS): Xcode and a Mac

[id="getting-started"]
== Getting Started

[id="add-dependency"]
=== 1. Add Dependency to Your Project

Add the following to your project's `build.gradle.kts`:

[source,kotlin]
----
kotlin {
    sourceSets {
        commonMain.dependencies {
            implementation("xyz.malefic:fidget:1.0.0")
        }
    }
}
----

[id="basic-usage"]
=== 2. Basic Usage

Import and use the library components in your Compose Multiplatform application:

[source,kotlin]
----
@Composable
fun MyApp() {
    App()
}
----

[id="project-structure"]
== Project Structure

Fidget is a Kotlin Multiplatform library with shared business logic and UI components for Android and iOS.

[source,plaintext]
----
Fidget/
├── shared/                                       # Kotlin Multiplatform library module
│   └── src/
│       ├── androidMain/                          # Android-specific code
│       │   └── kotlin/com/jetbrains/kmpapp/
│       ├── iosMain/                              # iOS-specific code
│       │   └── kotlin/com/jetbrains/kmpapp/
│       │       └── MainViewController.kt         # iOS ViewController for Compose UI
│       └── commonMain/                           # Shared Kotlin code (runs on both Android/iOS)
│           └── kotlin/com/jetbrains/kmpapp/
│               ├── App.kt                        # Main Composable, root of the UI
│               └── screens/
│                   ├── EmptyScreen.kt            # Example screen
│                   └── DemoScreen.kt             # Example screen
├── build.gradle.kts                              # Root build configuration
└── README.adoc                                   # This file
----

[id="building"]
== Building the Library

To build the library locally:

[source,sh]
----
./gradlew :shared:build
----

To generate documentation:

[source,sh]
----
./gradlew dokkaHtmlMultiModule
----

The documentation will be generated in `build/dokka/html/`.

[id="documentation"]
== Documentation

API documentation is generated using Dokka and can be built with:

[source,sh]
----
./gradlew dokkaHtmlMultiModule
----
